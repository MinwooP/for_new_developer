## 연습문제1 - N과 M

백트래킹 일반적인 코드 형태이니 잘 알아둘 것 ! 

```c++
int n, m; // 1~n 까지의 수 중에 m개를 고른 수열
int seq[10]; // 수열을 저장
bool isused[10]; //  각 index의 숫자가 이전에 아직까지의 수열에서 사용되었는지

void f(int k) { // seq[k]를 정하는 것.
	if (k == m) { // 이미 m개가 정해져 있을 것  
		// 수열 출력
		for (int i = 0; i<m ; i++) {
			cout << seq[i] << ' ';
		}
		cout << "\n";
		return;
	}

	for (int i = 1; i <= n; i++) {
		if (isused[i]) continue;

		seq[k] = i;
		isused[i] = true;
		f(k + 1); // 이제 다음 수를 고를 차례
		isused[i] = false;
	}
}

	cin >> n >> m;
	f(0);
```
